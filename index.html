Variables and Data types

A variable is a name that stores a value, and a data type defines the kind of value a variable holds.

Common Data Types in Python:

  • String (str) → Text data → "Hello"
  • Integer (int) → Whole numbers → 10
  • Float (float) → Decimal numbers → 3.14
  • Boolean (bool) → True or False

Examples:

# Input:
name = "Alice"
age = 25
height = 5.7
is_student = True

# Output:
print("Name: ", name)
print("Age: ", age)
print("Height: ", height)
print("Is a student: ", is_student)
Result:
Name: Alice
Age: 25
Height: 5.7
Is a student: True
Note: name, age, height and is_student are what you call variables.

Conditional Statements

Conditional statements allow a program to make decisions based on conditions. The most common ones are if, elif, and else.

Examples:

# Input:

if age >= 18:
    print("You are an adult.")
elif age >= 13:
    print("You are a teenager.")
else:
    print("You are a child.")

# Output:
It will ask for your age.
Enter your age: |
Example: 15

It will return.
You are a teenager.

int(input()) is an example of a typecasting.

Loops

Loops help repeat code multiple times. The two main types of loops in Python are for and while.

  • for Loop (Repeats a set number of times).
  • while Loop (Repeats while a condition is True)

Examples: (for loop)

# Input:
for i in range(5):
    print("Hello!")

# Output:
Hello!
Hello!
Hello!
Hello!
Hello!

Examples: (while loop)

# Input:
count = 0
while count < 3:
    print("Python is fun!")
    count += 1

# Output:
Python is fun!
Python is fun!
Python is fun!

Python Classes (Part 1)

What are Classes?

A class is a blueprint for creating objects. It defines a set of attributes and methods that the objects of that class will have.

Key Terminology

  • Object: An instance of a class
  • Attribute: A variable that stores data within a class
  • Method: A function defined inside a class
  • Constructor (__init__): A special method that initializes a new object
  • Self: A reference to the current instance of the class

Basic Class Structure

class Pet:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def introduce(self):
        print(f"Hi! I'm {self.name}, {self.age} years old")

Creating and Using Objects

# Create a pet object
my_pet = Pet("Buddy", 3)

# Use object methods
my_pet.introduce()
# Output: Hi! I'm Buddy, 3 years old

Python Classes (Part 2)

Inheritance

Classes can inherit attributes and methods from other classes. This is called inheritance.

Inheritance Example

class Animal:
    def __init__(self, name, species):
        self.name = name
        self.species = species

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name, species="Dog")
        self.breed = breed
    
    def bark(self):
        print(f"{self.name} says Woof!")

Using Inheritance

# Create a dog object
my_dog = Dog("Max", "Golden Retriever")

# Access inherited attributes
print(f"Name: {my_dog.name}")
print(f"Species: {my_dog.species}")
print(f"Breed: {my_dog.breed}")
my_dog.bark()

# Output:
# Name: Max
# Species: Dog
# Breed: Golden Retriever
# Max says Woof!
Why Use Classes?
  • Organize and structure code
  • Create reusable and modular code
  • Model real-world entities and their behaviors
  • Implement object-oriented programming principles
